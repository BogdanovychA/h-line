# Як долучитися до проєкту

Я вітаю будь-який внесок, що допоможе цифровізації державних послуг в Україні!

### 💡 Пропозиції та звіти
1. Перевірте список [Issues](https://github.com/BogdanovychA/h-line/issues), чи проблема вже не описана.
2. Якщо ні — створіть новий Issue з чітким описом.

### 💻 Робота з кодом
1. Зробіть **Fork** репозиторію.
2. Створіть окрему гілку (`git checkout -b feature/awesome-stuff`).
3. Використовуйте [uv](https://github.com/astral-sh/uv) для встановлення залежностей.
4. Перед комітом відформатуйте код (`black` та `isort`).
5. Відправте **Pull Request**.

### ⚖️ Важливо
- Увесь внесений код розповсюджується на умовах ліцензії **EUPL-1.2**.
- Дотримуйтеся принципів чистих комітів та описуйте зміни.

Дякую за вашу допомогу!
